# GAIA Backend

FastAPI backend for the **GAIA Intelligent Health Symptom Checker** (Final Year Project).

## Tech Stack
- FastAPI
- SQLite (local database)
- SQLAlchemy
- Pydantic

## Purpose
This backend handles:
- Symptom assessment requests
- Machine learning inference
- Risk classification & explanation
- Local data storage

> This system is intended for educational purposes only and does not replace medical professionals.
